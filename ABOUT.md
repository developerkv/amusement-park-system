#  Online Amusement Park Management System

##  Description
This is a web-based application developed using PHP and MySQL to manage amusement park bookings and ticket system.

##  Features
- Ticket booking system
- Admin, Staff, Customer modules
- QR code generation and verification
- Ride management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

##  Setup Instructions
1. Install WAMP/XAMPP
2. Import database.sql in phpMyAdmin
3. Place project folder in htdocs
4. Run in browser using localhost

##  Note
Project runs on local server environment
